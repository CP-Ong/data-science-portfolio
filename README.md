### Project 1: Early Warning Modelling for Running Injury Risk &nbsp; | &nbsp; [<ins>See Project →</ins>](Running-Injury-Risk)
> Developed an early-warning injury risk model using wearable training data to identify high-risk periods before injuries occur. The project focuses on transforming raw daily training metrics into interpretable injury risk scores that could be used in a consumer fitness or sports analytics product.

**Dataset:**  
Injury Prediction for Competitive Runners — Kaggle
